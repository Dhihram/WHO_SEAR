# WHO SEARO PHI

This is a repository of WHO SEARO, Public Health Intelligence, Health Emergency Information and Risk Assessment

This repository consists of :

| Folder | Name |
|:-------|:------|
| `Dat`    | The folder that contains data that generate from multiple sources |
|  `Misc`  | The folder that contains the logo, geojson, etc. |
|  `QIRA`  | The folder that contains features for QIRA Risk Assessment |
|  `climate` | Containing climate data for all countries in region |
| `script`  | Containing all of scripts |
